# luci-app-wolplus（网络唤醒++）

luci-app-wolplus 是一个用于 OpenWrt/ImmortalWrt 的网络唤醒 LuCI 插件，可更方便地管理局域网设备并发送 Wake-on-LAN 唤醒请求。
